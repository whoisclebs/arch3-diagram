# Flujo AI-first

## Flujo recomendado

1. escribe o ajusta la especificación del diagrama
2. valida el JSON con `@arch3/arch3-dsl`
3. renderiza con `@arch3/arch3-plantuml`
4. revisa el resultado con pruebas y documentación

## Beneficios

- contexto predecible para LLMs
- salida determinística
- evolución guiada por contratos
