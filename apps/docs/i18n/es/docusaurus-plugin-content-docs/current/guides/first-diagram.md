# Tu primer diagrama

El editor busca cumplir un rol parecido al de Swagger Editor: escribir la
fuente, validarla rápido y aprender la notación con feedback inmediato.

1. ejecuta `npm install`
2. ejecuta `npm run dev:editor`
3. abre el ejemplo `commerce.arch3.json`
4. elige la capa que quieres inspeccionar
5. previsualiza el render actual mientras ajustas la notación
