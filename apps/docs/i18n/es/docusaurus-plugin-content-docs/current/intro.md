# arch3-diagram

arch3-diagram es una metodología de diagramación de arquitectura inspirada en
el C4 Model.

Es independiente de PlantUML. PlantUML es solo un destino de render mediante
una extensión de Arch3.

Se enfoca en tres capas:

- contexto
- contenedores
- componentes

## Qué incluye esta documentación

- la línea base de la notación Arch3
- el modelo canónico para la versión 0.1
- el contrato visual para futuros renderizadores
- el papel del editor de referencia y de las extensiones de render
- documentación de referencia multilingüe

## Qué sigue

- un primer renderer visual compatible para PlantUML
- una DSL más rica además de JSON
- asistentes de IA para generación, revisión y mantenimiento
