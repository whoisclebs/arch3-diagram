# arch3-diagram

arch3-diagram é uma metodologia de diagramação de arquitetura inspirada no C4
Model.

Ela é independente de PlantUML. PlantUML é apenas um destino de renderização
por meio de uma extensão do Arch3.

Ela foca em três camadas:

- contexto
- containers
- componentes

## O que esta documentação inclui

- a linha de base da notação Arch3
- o modelo canônico para a versão 0.1
- o contrato visual para futuros renderizadores
- o papel do editor de referência e das extensões de renderização
- documentação de referência multilíngue

## O que vem depois

- um primeiro renderer visual compatível para PlantUML
- uma DSL mais rica além de JSON
- assistentes de IA para geração, revisão e manutenção
