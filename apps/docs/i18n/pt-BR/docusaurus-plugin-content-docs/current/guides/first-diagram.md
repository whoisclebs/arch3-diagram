# Seu primeiro diagrama

O editor busca cumprir um papel parecido com o do Swagger Editor: escrever a
fonte, validá-la rápido e aprender a notação com feedback imediato.

1. execute `npm install`
2. execute `npm run dev:editor`
3. abra o exemplo `commerce.arch3.json`
4. escolha a camada que deseja inspecionar
5. visualize o render atual enquanto ajusta a notação
