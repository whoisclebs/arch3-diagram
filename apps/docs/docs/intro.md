# arch3-diagram

arch3-diagram is an architecture diagramming methodology inspired by the C4
Model.

It is independent from PlantUML. PlantUML is only one rendering target through
an Arch3 extension.

It focuses on three layers:

- context
- containers
- components

## What this documentation includes

- the Arch3 notation baseline
- the canonical model for version 0.1
- the visual contract for future renderers
- the role of the reference editor and rendering extensions
- multilingual reference documentation

## What comes next

- a first compliant visual renderer for PlantUML
- a richer DSL beyond JSON
- AI assistants for generation, review, and maintenance
